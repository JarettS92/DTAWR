Overview
Gives step by step instructions on how to use these tools and reports

WARNING: Only for use by Dynatrace employees. Do not distribute to customers.


This section is to tell the user how to go about using the Reports functions

1. Make sure you configure your Tenants first! (top right corner)

2. Navigate to the report you want to run

3. Fill in any required data (dates, tags)

4. Run the report **Some may download a .csv file on completion**
